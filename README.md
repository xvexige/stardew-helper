# 星露谷农业管理速查手册设计需求

## 一、核心目标
开发一个集成化的星露谷生产管理工具，重点实现酿酒与种植的时间规划与提醒功能，帮助玩家优化农场运营效率。

## 二、日历系统设计
### 1. 主日历界面
- **结构**：4×7的28天网格，对应星露谷四季（春、夏、秋、冬）
- **视觉编码**：
  - 不同颜色圆点/图标区分事件类型：
    - 🔴 收酒提醒
    - 🟢 收菜提醒
    - 🟡 特殊事件/商人
    - 🔵 献祭截止日
  - 文字标注具体时间点（如“AM6:00收酒”）
- **交互**：每个日期格可点击进入详情页

### 2. 详情页功能
点击日期后显示：
- 当天所有待办事项列表
- 可勾选完成的事项
- 添加临时备注功能

## 三、酿酒管理模块
### 1. 初始设置
```
【添加酿酒批次】
- 选择酒桶位置（下拉菜单）：
  ◻️ 农舍内
  ◻️ 农场户外
  ◻️ 地窖
  ◻️ 自定义区域（可命名）
- 选择酿造物品（仅显示当前季节可酿造物）
- 设置开始时间（默认当前游戏日）
- 系统自动计算完成时间：
  基础时间 × 酿造技能加成 = 实际完成时间
```

### 2. 显示规则
- 周期性酿造（如啤酒花每天收）显示连续提醒
- 进度条显示酿造进度（如：3/7天）
- 可批量管理同一位置的酒桶

## 四、种植管理模块
### 1. 作物设置
```
【添加种植计划】
- 选择作物种类（仅显示当前季节作物）
- 设置种植日期
- 施肥选项：
  ◻️ 无肥料
  ◻️ 基础肥料（-10%时间）
  ◻️ 高级生长激素（-25%时间）
  ◻️ 顶级肥料（-33%时间）
- 系统自动计算：
  基础成熟时间 × (1 - 肥料加成) = 实际成熟时间
```

### 2. 收获提醒规则
- 单次收获作物：仅显示一次收获日
- 多次收获作物：
  - 首次收获后自动生成后续周期提醒
  - 可设置最大收获次数或结束日期
- 可调节因素自动计算：
  - 农业学家职业（-10%生长时间）
  - 生长激素叠加效果

## 五、特殊事件集成
### 1. 固定事件（自动添加）
- 季节性商人出现时间
- 节日日期
- 生日提醒（可配置关注NPC）
- 旅行货车日（每周五、日）

### 2. 收集品追踪

- 博物馆捐赠记录
- 烹饪食谱收集
- 鱼王钓获状态
