# 星露谷物语助手里日历上应该显示的信息

## 1. 任务提醒

### 种植收获提醒
- **来源**：`src/data/plannerData.js` 中的 `harvest` 类型任务
- **显示内容**：
  - 作物名称（如：防风草、草莓等）
  - 收获标记（绿色背景）
  - 地点信息（如有指定）
  - 肥料信息（如有使用）
  - 职业加成（如有选择）
  - 生长进度条
- **触发条件**：
  - 单次收获作物：种植后经过计算的成熟天数
  - 多次收获作物：首次收获后，按收获周期重复提醒

### 酿酒完成提醒
- **来源**：`src/data/plannerData.js` 中的 `wine` 类型任务
- **显示内容**：
  - 酿造品名称（如：啤酒、果酒等）
  - 完成标记（红色背景）
  - 酿造地点（如：农舍内、地窖等）
  - 进度条
- **触发条件**：酿造开始后经过设定的酿造天数

## 2. NPC生日

- **来源**：`src/data/calendarEvents.js` 中的 `birthdayData`
- **显示内容**：
  - 生日标记 🎂
  - 鼠标悬停显示NPC姓名
- **包含的NPC**：肯特、刘易斯、克林特、文森特、阿比盖尔、海莉、潘妮、山姆、艾利欧特、塞巴斯蒂安、皮埃尔、艾米丽等

## 3. 节日活动

- **来源**：`src/data/calendarEvents.js` 中的 `festivalData`
- **显示内容**：
  - 节日标记 🎉
  - 鼠标悬停显示节日名称
- **各季节节日**：
  - **春季**：复活节（13日）、花舞节（15日）、蛋蛋节（24日）
  - **夏季**：夏威夷宴会（11日）、月光水母节（28日）
  - **秋季**：星露谷展览会（16日）、万灵节（27日）
  - **冬季**：冰雪节（8日）、冬日星盛宴（25日）

## 4. 特殊天气

- **来源**：`src/data/weather.js` 中的 `getSpecialWeather` 函数
- **显示内容**：
  - 天气图标（如雷雨 ⛈️、节日天气 🎉）
  - 鼠标悬停显示天气类型和原因
- **特殊天气情况**：
  - 夏季13日和26日必出雷雨
  - 所有节日当天强制晴天

## 5. 特殊鱼类

- **来源**：`src/data/specialFish.js` 中的 `allSpecialFish`
- **显示内容**：
  - 鱼类标记 🐟
  - 鼠标悬停显示"特殊鱼类"
- **特殊鱼类包括**：
  - **春季**：鲶鱼（雨天河流）、西鲱（雨天河流）、鳗鱼（雨天海洋）
  - **夏季**：河豚（晴天海洋，特定时间）
  - **秋季**：大眼鱼（雨天淡水区）
  - **冬季**：冰川鱼（晴天矿井池塘）

## 信息优先级和显示规则

1. **位置布局**：
   - 日期数字左上角显示
   - 生日标记右上角
   - 节日标记左上角（节日标记优先于日期数字显示）
   - 特殊天气标记左下角
   - 特殊鱼类标记右下角
   - 任务列表显示在日期格子下方，按时间顺序排列

2. **颜色编码**：
   - 种植收获任务：绿色背景
   - 酿酒任务：红色背景
   - 节日标记：黄色背景

3. **交互功能**：
   - 点击日期格子可进入详情页
   - 任务条目可鼠标悬停查看详细信息
   - 任务条目支持删除操作

## 数据更新和同步

- 所有日历数据保存在本地 `localStorage` 中
- 任务添加/删除时自动保存
- 自定义地点信息独立保存
- 页面刷新后数据持久化

## 可扩展的信息

未来可以考虑添加的日历信息：

1. **商店营业时间**：特定商店关闭/特殊营业时间
2. **特殊物品刷新**：如旅行货车、科罗布斯商店物品
3. **献祭包截止日期**：提醒完成献祭包收集
4. **技能升级提示**：提醒技能升级时间点
5. **动物产品收集**：动物产品收获提醒
6. **树木果实成熟**：果树结果提醒

以上是星露谷物语助手日历系统当前显示和未来可以考虑添加的所有信息类型。
